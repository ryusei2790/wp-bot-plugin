Dify AIと連携するチャットボットウィジェット。ボタン・ビュワーの位置と色をカスタマイズ可能。

== Description ==

Dify Chat Botは、Dify AIプラットフォームと連携するWordPress用チャットボットプラグインです。

**主な機能:**

* Dify Chat APIとの連携（APIキーとURLで簡単設定）
* フローティングチャットボタンの表示
* チャットビュワー（会話ウィンドウ）の表示
* ボタンとビュワーの画面表示位置をカスタマイズ（上下左右＋オフセット）
* ボタンとビュワーの色をカスタマイズ（カラーピッカー付き）
* APIキーはサーバー側で安全に管理（フロントエンドに露出しない）
* レスポンシブ対応（モバイルでも快適に使用可能）

== Installation ==

1. `dify-chat-bot` フォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面の「プラグイン」で有効化
3. 「設定」→「Dify Chat Bot」からAPI URLとAPIキーを設定
4. ボタン・ビュワーの位置と色をお好みで調整
